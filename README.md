**[ ПРОБЛЕМНОЕ ПОЛЕ ]**

Отсутствие единой и доступной всем членам семьи базы лекарств с указанием их срока годности и их назначения. Несвоевременное пополнение домашней аптечки, в связи с этим велик риски употребить просроченное лекарство в то время, когда оно очень нужно.  Ежедневной рутиной каждой семьи является приобретения продуктов питания и товаров первой необходимости. Часто случаются ситуации, когда нужные продукты никем не были куплены, но приобретено что-то другое в большем, чем нужно количестве, тем самым растрачивая семейный бюджет впустую. Из-за забывчивости и занятости члены семьи не соблюдают режим полива и удобрения или путают периодичность и способы полива для разных растений, в результате чего растения чахнут.

**[ ЗАКАЗЧИК / ПОТЕНЦИАЛЬНАЯ АУДИТОРИЯ ]**

Целевая аудитория: семьи, группы людей, объединенных бытовыми интересами.

* Школьники, студенты и их родители - подходит для людей любого возраста.
* Взрослые люди 20-50 лет, которые из-за занятости забывают о бытовых потребностях.

**[ ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ ]**

Приложение для личное/семейного использования. 
Не предназначено для крупных бизнесов или больших сообществ. 
Цифровой продукт, распространяемый через магазины приложений. 
Авторизация пользователей и функционал создания совместной (семейной) группы, добавление участников с правами. 
* Модуль 1. Умная аптечка. Добавление лекарства, заполнение карточки с указанием срока годности, уведомления об окончании срока годности. 
* Модуль 2. Совместные покупки. Регулярно обновляемый список продуктов и товаров первой необходимости. Добавление нового товара, назначение даты закупки товара, настройка периодичности добавления в список покупок, сортировка товаров.
* Модуль 3. Уход за растениями. Картотека растений: Для каждого цветка можно указать название, фото, особенности. График полива: Настраивается индивидуально для каждого растения. Оповещения. Ведение история ухода

**[ ПОХОЖИЕ / АНАЛОГИЧНЫЕ ПРОДУКТЫ ]**

Большинство семейных приложений направлены на контроль местоположения и телефона ребенка.
Остальные же - обычный календарь с приятными бонусами (например, фотоальбом или книгу семейных рецептов).
